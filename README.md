# Credit Scoring ML Model

## Project Overview
This project predicts an individual's creditworthiness using Machine Learning algorithms.

The model analyzes financial history such as:
- Income
- Loan Amount
- Employment Length
- Interest Rate
- Credit History

## Algorithms Used
1. Logistic Regression
2. Random Forest Classifier

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC

## Dataset
Credit Risk Dataset from Kaggle.

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Performance Comparison

## Results
Random Forest performed better compared to Logistic Regression with higher ROC-AUC and F1-Score.
